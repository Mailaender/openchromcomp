How to install OpenChrom?


Prerequisites:
	The JRE or JDK 1.6 (http://java.sun.com/javase/downloads/index.jsp) must be installed before running OpenChrom.
	JRE (actual: Java SE Runtime Environment 6u19)
	JDK (actual: JDK 6 Update 19)

Run OpenChrom:
	Unzip the downloaded OpenChrom version and run the native launcher.
	You don't need to install the software.
